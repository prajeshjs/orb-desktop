About Orb

Orb is a powerful desktop task management application that runs quietly in the background and keeps you on track with smart reminders, follow-ups, and an AI-powered assistant.

It is designed to sit in your system tray and provide timely notifications so you never miss a deadline.

Smart Reminders

Get notified before your task is due, with customizable reminder durations.

Follow-up System

Automatic follow-ups at configurable intervals until you complete your task.

AI Assistant

Create, edit, and manage tasks using natural language via the AI window.

Reports & History

Track your productivity with detailed reports and event history.

Carry-Over Tasks

Incomplete tasks automatically carry over to the next day.

Repeat Daily

Set tasks to automatically repeat every day.

User Guide
Learn how to use Orb effectively
Getting Started ▼
  1. Orb runs in the background after installation. Look for the Orb icon in your system tray (bottom-right).
  2. Use the keyboard shortcut Ctrl+Shift+T to open the task creation window.
  3. Fill in the task name, time, reminder duration, and priority, then click Save.
  4. Orb will remind you before the task is due and follow up if it is not completed.

Managing Tasks ▼
  1. Create: Open the task window (Ctrl+Shift+T), enter details, and save.
  2. Edit: Open task list (Ctrl+Shift+L), click the edit icon on a task.
  3. Complete: Click the complete button on a task in the list, or respond "Yes" in a follow-up popup.
  4. Delete: Click the delete button on a task in the list.
  5. Carry Over: When a follow-up appears, choose "Carry Over" to move the task to tomorrow.
Using the AI Assistant ▼

The AI assistant lets you manage tasks using natural language. Open it with Ctrl+Shift+A.

  1. Create a task: "Create a task called Review PR at 3 PM remind 15 minutes before in quadrant 1"
  2. Show tasks: "Show my tasks" or "Show pending tasks"
  3. Complete a task: "Complete Review PR"
  4. Delete a task: "Delete Review PR"
  5. Show report: "Show report" or "Show history"

You must set an API key in the API Key section to use the AI assistant.

How to Get an API Key ▼

Orb uses the Groq API for AI features. Here is how to get a free API key:

  1. Go to console.groq.com in your browser.
  2. Sign up for a free account or log in if you already have one.
  3. Navigate to API Keys in the left sidebar.
  4. Click "Create API Key" and give it a name (e.g., "Orb").
  5. Copy the generated API key.
  6. Go to the API Key section in this app (left sidebar) and paste the key, then click Save.
Notification System ▼
  1. Reminder: Shown before the task is due (based on your configured duration).
  2. Follow-up: After the due time, you will get periodic follow-ups asking if the task is completed.
  3. Pre-Alert: Shown 1 minute before a follow-up if the system was off earlier.
  4. System Message: Shown when the system was off and missed reminders/follow-ups.

Voice Assistant

Control Orb with your voice. Press Ctrl+Shift+V to activate voice commands. Speak naturally to create tasks, open windows, manage tasks, and more.

When enabled, the app can access your microphone only when you activate it. The mic is released immediately after processing your command. No audio is recorded or stored.

Enable Microphone Access
Allow Orb to use your microphone for voice commands. The mic is only active while the orb is rotating — never in the background.
Disabled — Voice commands are off
How to use:
1. Enable microphone access above
2. Press Ctrl+Shift+V to start listening
3. Orb starts rotating — speak your command
4. Orb responds with voice and performs the action
Note: You can single-click the orb to instantly stop or cancel the voice agent.

Example commands:
• "Create task credit the payment at 5pm today remind me 5 minutes before in quadrant 1"
• "Show my tasks" • "Open calendar" • "Delete review PR"
• "Complete morning standup" • "Close history window"
• "Change the theme to light mode" • "Show only my urgent tasks"
• "Turn on auto start" • "Snooze that notification for a bit"

Start with System

When enabled, Orb will launch automatically when your computer starts up. It runs silently in the background and shows a notification confirming it is active.

When disabled, you need to open the app manually (double-click or Start Menu). The Home window will appear on manual launch.

Auto-start on Boot
Launch Orb automatically when your computer powers on. The app runs in the background — open the Home page anytime from the system tray or app icon.

About the API Key

  • The API key is used only for the AI Assistant feature.
  • All other features (tasks, reminders, follow-ups, reports) work without an API key.
  • Your key is stored locally on your computer and is never sent anywhere except to the Groq API.
  • You can delete your key at any time using the button above.

Get a Free API Key

  1. Visit console.groq.com and create a free account.
  2. Go to API Keys in the dashboard.
  3. Click "Create API Key", name it, and copy it.
  4. Paste it in the field above and click Save.
