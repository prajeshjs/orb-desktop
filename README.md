**Orb** is a minimalist, professional desktop application that intelligently tracks, schedules, and manages your daily tasks. Designed for zero-friction productivity, it works silently in the background and elegantly organizes your work using natural language processing and automated carry-overs.

👉 **[Download Orb Desktop](https://orb-web-eta.vercel.app/)**

---

## ✨ Features

- **Instant Task Entry:** Quickly add tasks with smart natural language processing.
- **Eisenhower Matrix Integration:** Intelligently categorizes tasks into quadrants based on urgency and importance.
- **Smart Follow-ups:** Proactively checks in on you with discreet pop-ups when a task duration finishes.
- **Automated Carry-Overs:** Unfinished tasks automatically roll over to the next day at midnight.
- **Zero Distractions:** Stays quietly in your system tray and only appears when you need it.
- **Sleek Aesthetics:** Features a beautifully crafted, premium UI with full Dark and Light mode support.

---

## 🛠️ Built With

- **Electron & Node.js** - Core desktop framework and background processing.
- **Vanilla JS & CSS** - Lightweight, blazing-fast UI without bloated frameworks.
- **SQLite** - 100% local, secure storage for all your tasks (no cloud lock-in).

## 💻 Local Development

If you'd like to run Orb from source or contribute to the project:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/orb-task-assistant.git
   cd orb-task-assistant
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Run in development mode:**
   ```bash
   npm start
   ```

4. **Build the Windows executable:**
   ```bash
   npm run build
   ```
